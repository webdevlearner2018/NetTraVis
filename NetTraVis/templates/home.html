Introduction

The group chose network traffic visualization to display network data visually using graphs, someone has lack of knowledge of reading a network data file would be able to read the graphs easily and clearly.

The aim of this project was to develop an application that the users can upload a network traffic data file (pcap file), choose the data to be displayed onto visual graphs. Data can be displayed in different protocols, different source or destination IP addresses, different flags, etc. The users can also compare them against each other depending on what data the file contains.

This project was created for our third year group project for Digital Forensics and Cyber Security.
